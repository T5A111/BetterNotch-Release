# Better Notch

一款增強版 macOS 劉海工具，基於 [NotchDrop](https://github.com/Lakr233/NotchDrop) 打造，
新增分頁切換、完整剪貼簿管理、日曆整合等實用功能。

## 📥 下載安裝

### 最新版本
📦 **[下載 BetterNotch v2.0](BetterNotch.dmg)**

### 安裝步驟
1. 下載上方的 `.dmg` 檔案
2. 開啟 `.dmg` 檔案
3. 將 BetterNotch 拖曳到應用程式資料夾
4. 首次啟動時，依照系統提示授予必要權限（步驟如下）
   嘗試打開 App → 出現警告。
   打開 系統設定 → 隱私權與安全性。
   在最下方會出現一個選項「仍要打開 Better Notch」。
   點選後，就能執行。

### 系統要求
- macOS 14.0 或更高版本
- Intel 或 Apple Silicon Mac

## ✨ 功能特色

### 🗂️ 六大功能頁面
- **暫存櫃 (第一頁)**：拖曳檔案快速暫存，支援 AirDrop 分享與多檔案管理。
- **媒體播放 (第二頁)**：精美封面與歌曲資訊展示，支援進度與音量調校。
- **系統資訊 (第三頁)**：CPU 折線圖、記憶體 (RAM) 圓環進度與即時網路 (上傳/下載) 監控。
- **天氣資訊 (第四頁)**：即時定位與自動天氣查詢，顯示氣溫、體感溫度、風速與濕度，附有日夜氣象漸層背景。
- **剪貼簿 (第五頁)**：智慧剪貼簿歷史管理，橫向卡片佈局與即時懸停預覽。
- **行事曆 (第六頁)**：EventKit 深度整合，今日事件顯示，一鍵啟動系統日曆。
**可根據需求在設定中關閉不需要的頁面**

### 🎯 核心功能
- ✅ **分頁切換**：支援觸控板手勢（雙指滑動）。
- ✅ **動態標題**：根據當前頁面顯示對應中文標題。
- ✅ **拖放開啟**：拖曳檔案到劉海區域自動展開工具面板。
- ✅ **頁面持久化**：自動記錄並還原最後使用的分頁。
- ✅ **媒體播放拓展**：背景有音樂軟體播放時自動拓展顯示音樂封面。

### 📋 剪貼簿功能
- **智慧監控**：自動記錄系統剪貼板變化，保存最近 10 條記錄
- **橫向佈局**：一次顯示 10 個項目，48x60px 緊湊設計
- **即時預覽**：滑鼠懸停顯示完整內容（最多 50 字）
- **一鍵操作**：點擊複製，個別刪除功能
- **搜尋過濾**：項目超過 6 個時自動顯示搜尋框

### 📅 行事曆功能
- **EventKit 整合**：完整的系統日曆權限管理與事件讀取
- **今日事件**：實時顯示當天日程，最多 3 個事件加「更多」按鈕
- **多版本兼容**：支援 macOS 14.0+ 和舊版本 API
- **系統整合**：一鍵啟動系統日曆應用

## ⚙️ 權限設置

首次啟動時需要授予以下權限：
- **輔助功能**：用於全域手勢檢測
- **日曆存取**：用於顯示日曆事件（可選）
